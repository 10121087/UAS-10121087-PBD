- Composer
- Apache / Nginx

Untuk running project pertama kali dan tanpa kendala pada saat menampilkan localhost 
1. Install composer dengan cara “composer install” di direktori project. Bisa via terminal vscode.
2. ubah file “.env.example” menjadi “.env”
3. kemudian type di terminal visual studio code dengan kode ini “php artisan key:generate lalu “php artisan config:clear” dan “php artisan cache:clear”
4. Ketik “php artisan serve” untuk menjalankan server, kemudian ketik di web http://127.0.0.1:8000
